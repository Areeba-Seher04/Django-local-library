# Django-Public-Library

## Permissions

### Admin.
1.Admins have all rights

2.Only admin can add librarians

### Users can:
1.Create Account

2.Login

3.Access all authors

4.Access all books

5.Access all genres

6.Search books by clicking favorite genre

7.See their borrowed books

### Librarin can:
1.Also borrow books

2.Login

3.Access all authors

4.Access all books

5.Access all genres

6.Search books by clicking favorite genre

7.See their borrowed books

8.Add books

9.Add authors

10.Add genre

11.Add borrowers

12.See all borrowers

13.Renew book
